# Postwoman

A lightweight, simple Java-based API client and request builder designed to help test, debug, and interact with REST APIs.

---

## Features

- **HTTP Requests:** Send `GET`, `POST`, `PUT`, `DELETE`, and other HTTP requests effortlessly.
- **Header Management:** Custom headers and authentication configuration.
- **Request/Response Inspection:** Clear, formatted view of JSON/XML responses, HTTP status codes, and headers.
- **Clean Interface:** Lightweight and focused UI built for quick API testing.

---
